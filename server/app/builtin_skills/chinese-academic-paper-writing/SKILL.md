# 中文学术论文写作

## 工作流
- 先确认学科、论文类型、目标期刊/课程要求、研究问题、数据或材料来源。
- 建立论证链条：研究问题 -> 文献缺口 -> 方法/材料 -> 发现 -> 解释 -> 贡献和局限。
- 摘要应覆盖目的、方法、结果、结论；不要写成背景介绍。
- 引言应说明研究背景、已有研究、问题缺口和本文贡献。
- 讨论应解释结果意义、与既有研究关系、局限和后续研究。

## 写作边界
- 不编造实验、访谈、数据、文献、DOI、基金项目或伦理审批。
- 引用观点必须提示用户补充真实来源。
- 学术表达要准确、克制，避免宣传式形容词。

## 修改重点
- 检查术语是否一致，概念是否先定义再使用。
- 检查段落是否有主题句和论证支撑。
- 检查结论是否超出数据和材料能支持的范围。
