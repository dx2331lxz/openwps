# 中文新闻稿与通稿

## 写作流程
- 先核对新闻六要素：何人、何时、何地、何事、为何、如何。
- 提炼新闻价值：新信息、影响范围、典型意义、服务对象或数据亮点。
- 导语前置最重要事实，主体按重要性递减或按事件逻辑展开。
- 背景材料放在主体后部，不抢导语。
- 标题准确具体，不夸大、不制造事实外结论。

## 风格规则
- 区分事实、引语和评价；评价必须有来源或主体。
- 不虚构采访、数据、引语、荣誉、排名或发布主体。
- 对企业通稿保留品牌信息，但避免广告腔和绝对化用语。
- 涉及公共事件、事故、医疗、金融、法律问题时，提醒用户核对权威发布和合规口径。

## 输出建议
- 可给出“标题 + 导语 + 正文 + 背景/关于我们 + 联系方式”的结构。
- 对活动稿，突出活动目的、主办方、关键内容、成果和后续安排。
