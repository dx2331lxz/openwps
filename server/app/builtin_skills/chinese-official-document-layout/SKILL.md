# 中文公文版式

## 检查顺序
- 先确认文档类型是否属于党政机关公文；非正式公文只给“参照公文风格”的建议。
- 检查页面：A4 幅面、页边距、页码、版心稳定；不要为了塞内容牺牲可读性。
- 检查构成要素：份号、密级和保密期限、紧急程度、发文机关标志、发文字号、签发人、标题、主送机关、正文、附件说明、署名、成文日期、印章位置、附注、附件、抄送机关、印发机关和印发日期、页码。
- 检查标题：应概括事由并标明文种；避免标题过长、文种错用、多个文种并列。
- 检查正文层级：层次编号统一，段落间距稳定，附件说明和正文引用一致。
- 检查落款：发文机关署名、成文日期、印章位置和附件顺序相互对应。

## 修复原则
- 先修结构，再修细节；不要只改字体字号而忽略文种、主送、附件和落款不一致。
- 当用户需要严格出文时，提示以所在机关模板和正式标准文本为准。
- 对无法确认的涉密、紧急、印章、发文字号信息，用占位符或待核项，不自行生成。

## 输出格式
- 给出“发现的问题”“建议修改”“需要用户补充的信息”三部分。
- 如果要直接改文档，先保留原文事实，只调整结构、标题、层级和版式表达。

## 来源索引
- GB/T 9704-2012《党政机关公文格式》：党政机关公文版式标准。
- 《党政机关公文处理工作条例》：公文格式组成、公文用纸和拟制审核要求。
