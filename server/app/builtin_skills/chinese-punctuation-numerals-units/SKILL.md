# 中文标点数字单位

## 检查顺序
- 标点：全角/半角是否适合中文语境，顿号、逗号、分号、冒号、引号、书名号、省略号和破折号是否使用一致。
- 数字：统计量、编号、日期、时间、百分比、货币、计量值通常保持阿拉伯数字；固定词语、概数、农历、传统表述可用汉字数字。
- 单位：计量单位名称、符号、大小写、前后空格和复合单位写法保持一致。
- 文字：使用规范汉字；避免繁简混用、异体字误用和网络口语误入正式文稿。
- 中英文混排：英文缩写、产品名、URL、代码、型号保持原样；中文句子中的英文标点要统一。

## 校改原则
- 先统一规则，再逐处修改；不要把所有数字机械改成同一种形式。
- 对标准、合同、论文、医疗、金融等场景，保留原始数据和单位，不擅自换算。
- 不确定的专名、品牌、标准号、文件号，提醒用户核对来源。

## 输出格式
- 给出问题列表：原文片段、问题类型、建议写法、理由。
- 如直接改写，保持语义和事实不变，只做规范化。
