# GB/T 7714 参考文献

## 先确认版本
- 默认按 GB/T 7714-2015 处理，除非用户或机构要求其他版本。
- GB/T 7714-2025 已发布，实施日期为 2026-07-01；当前日期之前应提示用户确认单位是否提前采用。

## 工作流
- 识别引用制式：顺序编码制、著者-出版年制或学校/期刊自定义格式。
- 检查正文引用是否与文后条目一一对应。
- 按文献类型补齐作者、题名、载体/来源、出版项、年卷期页码、DOI/URL/访问日期等必要信息。
- 不确定信息用待补齐标记，不猜 DOI、页码、期刊卷期、出版社。
- 统一标点、空格、作者数量显示、英文大小写和中英文混排。

## 常见问题
- 文后有条目但正文未引用，或正文引用编号不存在。
- 同一作者/题名写法不一致。
- 网页资源缺少发布日期、更新日期或访问日期。
- 学位论文、会议论文、标准、专利、报告等类型混用期刊格式。
