# 中文学位论文格式

## 检查顺序
- 先询问或读取学校模板；学校要求优先于通用建议。
- 检查组成：封面、原创性/授权声明、中文摘要、英文摘要、目录、正文、参考文献、附录、致谢等是否符合要求。
- 检查层级：章、节、条标题编号连续，标题样式一致。
- 检查摘要：研究目的、方法、结果、结论齐备，关键词数量和分隔方式符合要求。
- 检查图表：编号、题名、来源、单位、注释、正文引用一致。
- 检查参考文献：正文引用与文后条目互相对应。

## 修复原则
- 不替用户编造文献、数据、实验或学校规定。
- 对格式要求冲突处，以用户给出的学校模板或学院通知为准。
- 输出时区分“可直接修改的问题”和“需要查学校规范的问题”。
