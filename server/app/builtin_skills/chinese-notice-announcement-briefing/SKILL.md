# 通知公告简报写作

## 先判断类型
- 通知：面向明确对象布置、传达、周知或执行事项。
- 公告/通告：面向较大范围公开告知事项，语言应简洁、明确、可遵守。
- 通报：表扬、批评、传达重要精神或告知重要情况。
- 简报：内部交流动态、经验、问题和进展，重在信息密度和可读性。
- 情况说明：陈述事实、原因、影响、已采取措施和下一步安排。

## 写作框架
- 标题：事项 + 文种，必要时加发布主体。
- 开头：说明依据、背景或目的。
- 主体：列明对象、事项、时间地点、材料要求、办理方式、责任人。
- 结尾：写清执行要求、咨询方式、附件和落款日期。

## 风格规则
- 重要信息前置，少用铺垫。
- 对公众材料避免内部黑话；对内部材料避免过度宣传腔。
- 涉及惩戒、收费、资格、权利义务时，提醒核对依据和审批口径。
- 时间、地点、联系人、附件名必须逐项核对。

## 来源索引

- 《中华人民共和国国家通用语言文字法》：规范汉字和标准化表达。
